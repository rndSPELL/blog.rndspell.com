Byzanz - легковесная и быстрая утилита для создани скринкастов.

Дата Пт. 08 Август 2014 Тэги byzanz / скринкасты

Когда мне потребовалас программа для захвата экрана, я искал максимально простую утилиту, легкую, желательно консольную. В репозиториях ubuntu я обнаружил byzanz.

Установка в ubuntu проста:

sudo apt-get install byzanz

Утилита состоит из двух: это byzan-record и byzanz-playback. Первое это запись экрана в формате GIF, OGG, FLV и т.д. второе - конвертация результата захвата в формате .byzanz в другой доступный формат.

byzanz-record имеет следующий синтаксис:

byzanz-record [ключи] путь/до/выходного/файла.[gif, ogg, ogv, flv, webm, byzanz]

расширение файла определяет формат записи.

Доступные ключи:

-a, --audio
      Запись аудио со стандартного устройства аудиоввода (микрофона).

-c, --cursor
      Записывать курсор

-d, --duration=SECS
      Длительность записи (по умолчанию: 10 с)

-e, --exec=COMMAND
      В отличии от ключа -d длительность записи будет определяться временем выполнения команды COMMAND

--delay=SECS
      Задержка старта записи (по умолчанию: 1 с)

--display=DISPLAY
      Использовать X DISPLAY

-h, --height=PIXEL
      Высота области захвата

-v, --verbose
      подробный вывод

-w, --width=PIXEL
      Ширина области захвата

-x, --x=PIXEL
      X координата области захвата

-y, --y=PIXEL
      Y координата области захвата

Пример использования:

byzanz-record -c -w 300 -h 200 -x 0 -y 30 --delay=5 file.gif
